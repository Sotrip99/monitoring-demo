# WebRTC 모니터링/미러링 데모

React + TypeScript 기반의 WebRTC 화면 공유 데모 애플리케이션입니다.

## 기능

### 1. 모니터링 기능 (학생 → 교사)
- 학생이 화면 공유를 시작하면 상단에 상태 칩 표시
- 교사가 학생 화면을 요청하면 학생에게 모달로 안내
- 학생이 허용하면 화면 공유 시작
- 공유 중단 시 자동 감지 및 상태 업데이트

### 2. 미러링 기능 (교사 → 학생)
- 교사가 화면 공유를 시작하면 상단에 상태 칩 표시 (공유 중지 버튼 포함)
- 학생 쪽에 교사 화면 뷰어 자동 표시
- 3가지 뷰 모드: 전체창 / 플로팅 / 최대화
- Picture-in-Picture (PiP) 지원
- 오디오 제어 (볼륨, 음소거)
- 공유 종료 시 자동 처리

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 사용 방법

1. 브라우저에서 `http://localhost:3000` 접속
2. 좌측: 교사 패널, 우측: 학생 패널
3. **모니터링 모드**: 교사 패널에서 "학생 화면 요청" 버튼 클릭
4. **미러링 모드**: 교사 패널에서 "화면 공유 시작" 버튼 클릭

## 주의사항

- Chrome 브라우저 권장
- 실제 WebRTC 연결을 위해서는 시그널링 서버(WebSocket)가 필요합니다
- 현재 구현은 데모용으로, 같은 브라우저 내에서 두 패널이 동작합니다
- 실제 배포 시에는 별도의 시그널링 서버를 구현해야 합니다
